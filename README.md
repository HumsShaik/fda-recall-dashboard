## 📌 Project Overview

This project analyzes FDA recall datasets to help identify:
- Companies with highest recall counts
- Common reasons for product recalls
- Dataset-wide trends and summaries

It also includes a **smart query assistant** that allows users to ask natural-language-like questions about the dataset.

---

## 🎯 Key Features

- 📂 Upload any FDA recall CSV dataset
- 📊 Automatic data analysis and insights
- 🏢 Top companies/firms with most recalls
- ⚠️ Most common recall reasons
- 🤖 Smart query assistant (rule-based NLP)
- 📈 Interactive charts and visualizations
- 🧠 Dynamic column detection (works with different datasets)
